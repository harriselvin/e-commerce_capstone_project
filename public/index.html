Home Page

HTTP METHOD PATH DESCRIPTION
USER INFORMATION
GET /users Gets all users
GET /user/:id Gets a single user by ID
POST /register Adds a new user
PATCH /user/:id Edit a user's information by ID
POST /login Logs a user in
DELETE /user/:id Removes a user by ID
ITEM INFORMATION
GET /items Gets all items/products
GET /item/:id Gets a single item by ID
POST /items Adds a new item
PUT /item/:id Edit a item's information by ID
DELETE /item/:id Removes a item by ID
CART INFORMATION
GET /user/:id/carts Gets all cart items
POST /user/:id/cart Adds a new item to cart
PATCH /user/:id/cart/:prodID Edit a cart item's information by ID
DELETE /user/:id/cart Removes a single cart item by ID
DELETE /user/:id/cart/:cartId Removes all cart items by
FAQ INFORMATION
GET /faq Gets all FAQ info
